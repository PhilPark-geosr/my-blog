# 기본 설정

기본 설정 방법을 알아봅시다.

## 설정 파일

설정은 `docusaurus.config.ts` 파일에서 관리됩니다.

## 주요 설정 항목

- **title**: 사이트 제목
- **tagline**: 사이트 태그라인
- **url**: 프로덕션 URL
- **baseUrl**: 기본 URL 경로

## 예시

```typescript
const config = {
  title: 'My Site',
  tagline: 'Dinosaurs are cool',
  url: 'https://example.com',
  baseUrl: '/',
};
```
